# Sprint 01: Modern JavaScript for React

## Overview
This sprint focuses on the advanced JavaScript concepts and ECMAScript (ES6+) features that form the foundation of modern React development. The goal was to move beyond basic syntax and understand the underlying mechanics of JS, as well as adopt functional programming paradigms essential for writing React components.

---

## Learning Materials

### 1. Modern Syntax & Data Handling
*   **Variable Declarations:** Deep dive into scope differences between `let`, `const`, and `var`.
*   **Destructuring Assignment:** Efficiently extracting data from arrays and objects (crucial for React Props and State).
*   **Spread & Rest Operators (`...`):** Copying objects, concatenating arrays, and handling indefinite function arguments.
*   **Template Literals:** Advanced string interpolation.

### 2. Advanced Function Mechanics
*   **Arrow Functions:** Syntax specifics and lexical `this` binding.
*   **Hoisting & Scope:** Understanding the execution context and memory allocation phase in JavaScript.
*   **IIFE (Immediately Invoked Function Expressions):** Data privacy and avoiding global scope pollution.
*   **Default Parameters & `arguments` Object:** Flexible function signatures and handling missing arguments.
*   **Function Overloading:** Strategies for implementing dynamic function behavior in JavaScript.

### 3. Functional Programming Foundations
*   **Pure Functions:** Writing functions without side effects, a core requirement for React functional components and Redux reducers.
*   **Function Composition:** Building complex logic by combining simpler functions.
*   **Array Methods:** Extensive practice with higher-order functions (`map`, `filter`, `reduce`, `find`, `some`, `every`) for data transformation and list rendering in React.
